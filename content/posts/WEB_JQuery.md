## jQuery 知识总结

jQuery 是一个 JavaScript 库。jQuery 极大地简化了 JavaScript 编程。



### $符号

`$`是著名的 jQuery 符号。实际上，jQuery 把所有功能全部封装在一个全局变量`jQuery`中，而`$`也是一个合法的变量名，它是变量`jQuery`的别名。
